# Branding Assets

NXGS Gaming PNG, ICO, ICNS, Android, Steam, official-artwork, and Switch branding assets have been generated from the supplied `Banner game.png` source image, using the NXGS Gaming wordmark area and excluding the PlayStation-style controller/logo area. SVG files are lightweight NXGS vector fallbacks.

- `gui/res/nxgs-gaming.png`
- `gui/res/nxgs-gaming-logo.png`
- `gui/res/nxgs-gaming-logo-white.png`
- `docs/images/nxgs-gaming.svg`
- `docs/images/nxgs-gaming-logo-white.svg`
- `docs/images/nxgs-gaming.png`
- `docs/images/nxgs-gaming-logo-white.png`

Review these generated files before publishing final builds.

The repository still contains upstream chiaki-ng/Chiaki artwork and platform assets for attribution, build compatibility, Steam shortcut generation, and historical documentation. Do not present upstream artwork as the NXGS Gaming logo unless the license permits that use and attribution is kept.
